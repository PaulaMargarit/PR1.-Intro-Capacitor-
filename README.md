## Tecnologies utilitzades
- p5.js

## Instal·lació
1. Instal·la les dependències:
   npm install
2. Executa el projecte en mode desenvolupament:
   npm run dev
   
## Executar a Android (opcional)
1. Instal·lar dependències:
   npm install

2. Compilar:
   npx cap sync

3. Obrir Android Studio:
   npx cap open android
